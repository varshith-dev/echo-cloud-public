Data Collection Disclosures

We believe in radical transparency. This page discloses every piece of data collected when you visit a public OQENS Photos page, why we collect it, and exactly where it goes.

Last updated: June 2026
Overview

What pages collect data? Only public shared album pages — not private authenticated sessions

Data is collected only when a visitor opens a public shared album link (e.g. photos.echo.oqens.me/shared/<token>). This is analytics data collected to help OQENS and album owners understand how their shared content is being viewed.

This data is never collected during:

  • Private authenticated dashboard sessions
  • File downloads from the main dashboard
  • Account management pages
  • Any page behind a login wall
Who sees this data? Only the OQENS platform administrator (host) can view this analytics data from the admin console. Individual album owners cannot see visitor analytics. The data is not shared with or sold to any third parties.

How Data Is Collected

Collection mechanism A JavaScript beacon sent in the background

When you open a public shared album, a JavaScript script runs after the page loads. It collects the fields listed above and sends them in a single HTTP POST request to https://echo.oqens.me/api/analytics/collect.

  • The script does not use cookies for tracking.
  • The script does not use browser fingerprinting techniques (canvas, WebGL, font enumeration, etc.).
  • The request is non-blocking — it runs after the media loads and does not affect page performance.
  • The collection happens once per page load. No persistent tracking across sessions.
  • No tracking pixels, third-party analytics scripts, or ad networks are used.
IP Geolocation: Your IP address is looked up using ip-api.com, a third-party geolocation service. Your IP is sent to their API for location resolution. ip-api.com's own usage terms apply to this lookup. The result (country, city, ISP) is stored by OQENS; the raw lookup is not cached by ip-api.com for our usage.

Your Rights

Your privacy rights GDPR, CCPA, and general data subject rights

Depending on your jurisdiction, you may have the following rights regarding the data collected about you:

  • Right to Access: Request a copy of the data collected about your IP address / visit.
  • Right to Erasure: Request deletion of your visit records.
  • Right to Object: Object to the processing of your data for analytics purposes.
  • Right to Portability: Request your data in a machine-readable format (CSV).
To exercise any of these rights, contact us at support.oqens.me. Include your approximate IP address (you can find it at whatismyip.com) and the date(s) of your visit so we can locate and delete your records.

How to opt out Technical methods to prevent data collection

You can prevent data collection by:

  • Using a VPN or Tor — masks your real IP address and location.
  • Disabling JavaScript in your browser — the collection script requires JS to run.
  • Using a content blocker — blocking requests to echo.oqens.me/api/analytics/collect.
  • Using a privacy browser — browsers like Firefox with Enhanced Tracking Protection enabled may block the request.
If JavaScript is disabled, the page will still display the shared album media (photos/videos). Only analytics collection will be prevented.
