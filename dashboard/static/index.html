Developer Integration Guide

Overview

Welcome to the OQENS developer integration guide. This dashboard exposes a clean, multi-tenant object storage REST API that lets you read, upload, and delete assets programmatically.

Authentication is performed by passing your unique API Key in the X-API-Key request header. All programmatic operations occur securely inside your isolated tenant directory.

Programmatic API Reference

GET List Files

Retrieve list of all active files and sizes in your storage bucket.

curl -H "X-API-Key: YOUR_API_KEY" https://echo.oqens.me/api/bucket/list

POST Upload File

Upload a new file using multipart/form-data payload.

curl -X POST -H "X-API-Key: YOUR_API_KEY" -F "file=@/local/path/file.png" https://echo.oqens.me/api/bucket/upload

GET Download File

Retrieve file stream. Pass parameter attachment=true to force download.

curl -H "X-API-Key: YOUR_API_KEY" -o file.png "https://echo.oqens.me/api/bucket/download?key=file.png&attachment=true"

DELETE Delete File

Delete file from bucket by passing its key as JSON body.

curl -X DELETE -H "X-API-Key: YOUR_API_KEY" -H "Content-Type: application/json" -d '{"key":"file.png"}' https://echo.oqens.me/api/bucket/delete

GET Public CDN URL

Access publicly shared links on dl.oqens.me without headers. Append ?preview=true for browser rendering.

https://dl.oqens.me/YOUR_CLOUD_ID/file.png
